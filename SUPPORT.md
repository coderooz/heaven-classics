## How to Get Help

### For Questions

- Start a [GitHub Discussion](https://github.com/coderooz/heavens-classics/discussions)
- Tag your issue with the `question` label

### For Bugs

- Search existing [issues](https://github.com/coderooz/heavens-classics/issues) first
- Open a new issue with the `bug` template
- Include reproduction steps and environment details

### For Features

- Search [issues](https://github.com/coderooz/heavens-classics/issues) for similar requests
- Open a new issue with the `enhancement` template
- Explain the problem and proposed solution

### For Documentation

- Open an issue with the `documentation` template
- Specify the section that needs improvement
